# Inventory Monitoring App
A full-stack web app based on Rust (Axum), SQLx and React. This inventory management app allows the user to:
- Add products to a product database (with given dimensions, weight, product ID and name).
- Add products from the database to inventory, which tracks location and quantity.
- Stores customers and their corresponding orders in the database.
- Stores customers orders in the database.

And presents all this information in a nice easy-to-use UI.

This is my first full-stack app so this is really a learning exercise. This was inspired by work I did part-time over the summer in a warehouse for a global logistics company (whose software - I think - was terrible).
